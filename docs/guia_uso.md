# Guía de Uso de la Calculadora Python

Esta guía te explica paso a paso cómo **instalar, ejecutar y utilizar** la calculadora desarrollada en Python, con un enfoque en buenas prácticas de **DevSecOps**: seguridad, automatización y reproducibilidad.

---

## Requisitos Previos

Antes de comenzar, asegúrate de tener instalado:

- **Python 3.9 o superior**
- **Git**
- **pip** (gestor de paquetes de Python)
- (Opcional) **Docker** – para ejecución en contenedores

> Verifica tu versión de Python:
> ```bash
> python3 --version
> ```
